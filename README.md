Sales Performance Analysis
Project Overview

This project analyzes supermarket sales data to identify key business insights such as sales trends, profit distribution, and top-performing products.

The analysis is performed using SQL, Python, and Power BI to demonstrate end-to-end Data Analyst skills.

Objectives

Understand overall sales performance

Identify high-performing regions and product categories

Analyze customer and product trends

Build an interactive dashboard for business decision-making

Tools & Technologies

Python (Pandas, Matplotlib)

SQL

Power BI

Jupyter / Google Colab

Dataset Information

The dataset contains transaction-level sales records of a supermarket.

Each row represents a single purchase.

Key columns include:

Product Category

Region / City

Quantity

Unit Price

Total Sales

Date

Customer Type
